VERCEL DEPLOY — agar error aaye:
"Couldn't find any pages or app directory"

=== FIX 1 (sabse common) ===
Vercel Dashboard → Project → Settings → General → "Root Directory"
→ Edit → "comptech-website" select karo → Save → Redeploy

(Iska matlab: GitHub repo ka root "new project" hai, lekin Next.js app andar
"comptech-website" folder me hai.)

=== FIX 2 ===
Repo root pe "vercel.json" hai jo automatically yeh chalata hai:
  cd comptech-website && npm install
  cd comptech-website && npm run build
Agar tumne POORA "new project" folder GitHub pe push kiya hai, yeh build sahi
folder se chalega.

=== FIX 3 ===
Agar GitHub repo sirf "comptech-website" files ka hai (root me package.json + src/):
→ Root Directory KHALI rakho (repo root = app)
→ "src/app" folder Zaroor push ho — local: git add src && git commit && git push

Prebuild script agar fail kare to message me likhega: src/app missing
